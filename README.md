# MegaMente

Bem-vindo ao **MegaMente**! Este é um projeto unificado que junta quatro repositórios distintos em um único ecossistema poderoso para agentes de IA.

> **Aviso Importante sobre Direitos Autorais / Créditos**
> Todos os créditos, direitos autorais e licenças originais pertencem exclusivamente aos criadores dos respectivos projetos incluídos nesta compilação. O projeto MegaMente apenas une os repositórios em uma estrutura única para fins de integração e pesquisa.

Os projetos contidos aqui são:
- **gbrain**
- **mindmuxai-brain**
- **mi4uu-brain**
- **amnesia-no-more**

## 🚀 Como Instalar

O grande diferencial do **MegaMente** é que, ao invés de baixar e configurar cada projeto um por um, você baixa todos de uma só vez e os configura com um único comando.

### Passo 1: Baixar todos os projetos
Abra seu terminal ou linha de comando e faça o clone deste repositório:
```bash
git clone https://github.com/MessiahSantus/MegaMente.git
cd MegaMente
```

### Passo 2: Instalar e Configurar Tudo
Você pode usar uma das opções abaixo para tornar todos os projetos usáveis simultaneamente:

#### Opção 1: Via NPM (Recomendado)
Na raiz do repositório, basta executar o comando abaixo. Ele instalará todas as dependências dos projetos Node.js simultaneamente e executará os scripts de setup.
```bash
npm run install:all
```
*(Alternativamente, apenas `npm install` configurará os workspaces).*

#### Opção 2: Via PowerShell (Windows)
Criamos um script automatizado que instala as dependências Node.js via npm (para `gbrain` e `mi4uu-brain`) e executa os instaladores independentes dos outros submódulos (`amnesia-no-more` e `mindmuxai-brain`).
Abra o **PowerShell** na raiz e digite:
```powershell
.\install.ps1
```

## 🎮 Como Usar

O **MegaMente** instala o ambiente de todos os projetos simultaneamente, mas cada um deles possui uma função específica e pode ser usado de forma independente.

Após a instalação bem-sucedida, navegue até a pasta do projeto que você deseja utilizar e siga as instruções contidas no `README.md` interno dele. Por exemplo:
- Para usar a memória persistente, acesse a pasta `amnesia-no-more`.
- Para utilizar as inteligências centrais e motores de IA, acesse `gbrain`, `mi4uu-brain` ou `mindmuxai-brain`.

Consulte a documentação específica dentro da pasta de cada projeto para ver os comandos exatos de inicialização!

### 🤖 Ativando o MegaMente na sua IA
O MegaMente vem com uma Skill nativa embutida para Agentes IA (como Antigravity, Claude Code, etc). 
Ao abrir a pasta deste projeto no seu editor (VS Code), sua IA detectará os superpoderes automaticamente. Basta digitar `/megamente` no chat para ativar a memória persistente (amnesia-no-more) e acionar os motores de processamento analítico (gbrain, mi4uu-brain e mindmuxai-brain).
