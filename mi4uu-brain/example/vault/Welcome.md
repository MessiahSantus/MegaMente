# Welcome to **brain.md**

Your local-first second brain — markdown vault, real-time search,
semantic RAG, and a built-in MCP server so any AI agent can read,
search and write into your notes safely.

> [!tip] Five-minute tour
> Open the notes in the sidebar in this order:
> 1. [[Wikilinks]]
> 2. [[Embeds & Transclusion]]
> 3. [[Callouts]]
> 4. [[Math]]
> 5. [[Mermaid Diagrams]]
> 6. [[Code Highlighting]]
> 7. [[Tasks & Tables]]
> 8. [[Frontmatter & Aliases]]
>
> Then check out the **AI for Agents** folder:
> [[RAG (Semantic Search)]], [[MCP Server]], [[Folder Permissions]].

## What makes brain.md different

- **Local & private** — runs entirely on your machine. Vault is just
  a folder of `.md` files. No lock-in. ==Open in VSCode any time==.
- **Agent-native** — first-class MCP server over HTTP+SSE means
  Claude Desktop (or any MCP-compliant agent) can read, search and
  write your notes with per-folder permissions.
- **Semantic search built in** — LanceDB vector store + your choice
  of local (Xenova/bge-small) or remote (Ollama / OpenAI / anything
  OpenAI-compatible) embedder.
- **Real markdown** — Obsidian-flavor: wikilinks, embeds, callouts,
  math, mermaid, highlights, tasks, frontmatter, aliases.
- **Git-native** — every save is an autocommit. Full history,
  diff, restore, manual checkpoints.

## Try this

1. Press **⌘P** to open search.
2. Type "mcp" to see semantic hits.
3. Right-click any folder in the sidebar → **MCP permissions…**
4. Open [[brain.md launch]] in the Projects folder for a real-world note
   with tasks, links and a roadmap.
